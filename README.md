# archlinux-install
Install script for Arch Linux.

## Download:
``` bash
# wget https://github.com/channeladam/archlinux-install/archive/master.zip
# pacman -Sy unzip
# unzip master
# cd archlinux-install-master/VirtualBox
# chmod +x *.sh
```

## Run install.sh
``` bash
# ./install.sh
```

## Run configure.sh
``` bash
# ./configure.sh
```
